Portfoliosimulation

Die Portfoliosimulation ist für jedermann kostenlos nutzbar und soll unter Nutzung von Realdaten den Handel mit Wertpapieren erfahrbar machen.

Screening

Im einfachen Screeningmodus können hier die Benutzer in mittels vordefinierter Kriterien nach Wertpapieren suchen. Zusätzlich wird ein erweiterter Screeningmodus angeboten, der die Angabe benutzerdefinierte Ausdrücke für die Suche erlaubt. Dadurch können die Benutzer auch komplexe Formeln abbilden, die aus eigenen Strategien stammen.

Watches

Mittels Watches kann sich ein Benutzer über bestimmte Ereignisse benachrichtigen lassen. Im simplen Modus kann der Benutzer eine Aktie auswählen, für welches er einen Watch definieren will. Anschließend erstellt der Benutzer unter Verwendung einer begrenzten Anzahl an vordefinierten Kriterien eine Watchbedingung.

Börsenspiele

Institutionen können Börsenspiele mit individuellen Einstellungen, wie zum Beispiel erlaubte Wertpapiere, erstellen.
